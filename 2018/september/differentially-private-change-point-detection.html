Differential Privacy

Differential Privacy is a technique to bound the effect that a single data point can have on a specific aggregate computation.

The intution is that if you look at all data, you don't want to be able to remove an individual's data from the aggregate to determine the individual's data.

  • Post-processing: Any functions applied to the result of a computation from a DP algorithm is DP

Why laplace error?

exponential.  It makes the math nice.

Is there something to be gained by viewing this as a statistical problem?
